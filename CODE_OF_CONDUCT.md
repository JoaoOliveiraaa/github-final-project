# Code of Conduct

## Our Pledge
We as members, contributors, and leaders pledge to make participation in our project a harassment-free experience.

## Our Standards
Examples of behavior that contributes:
- Using welcoming language
- Respecting different viewpoints
- Accepting constructive criticism

## Enforcement Responsibilities
Maintainers are responsible for clarifying standards and taking corrective action.

## Scope
Applies to all project spaces and public spaces when representing the project.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Maintainers will follow fair and consistent actions.

## Attribution
This Code of Conduct is adapted from common open-source standards.
